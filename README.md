Libnet
======

A Cross-platform library for building c++ socket applications.
------

###  BUILDING


> #### GCC(Linux)
```

$ git clone https://github.com/Vssblt/libnet.git
$ cd LibNet
$ make
$ make test
```


> ### Visual Studio(Windows)
```

$ git clone https://github.com/Vssblt/libnet.git
$ cd LibNet
$ ./sln
```
