LibNet
======

A Cross-platform library for building c++ socket applications.
------

###  BUILDING


> #### GCC(Linux)
```

$ git clone https://github.com/Vssblt/LibNet.git
$ cd LibNet
$ make
$ make test
```


> ### Visual Studio(Windows)
```

$ git clone https://github.com/Vssblt/LibNet.git
$ cd LibNet
$ ./sln
```


###  Remarks


> Please compile this project with a Windows SDK that supports C++ 11.
