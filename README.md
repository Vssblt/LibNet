LibEDTP
======

A Cross-platform library for building c++ socket applications.
------

###  BUILDING


> #### GCC(Linux)
```

$ git clone https://github.com/Vssblt/libedtp.git
$ cd libedtp
$ make
$ make test
```


> ### Visual Studio(Windows)
```

$ git clone https://github.com/Vssblt/libedtp.git
$ cd libedtp
$ ./sln
```
